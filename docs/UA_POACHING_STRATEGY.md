## 1. Карта ринку і наша позиція

Три сегменти:

| Сегмент | Апки | Чим б'ємо |
|---|---|---|
| **Прості ручні трекери** | Monefy, Orange dog, Wagenleitner, Sumlio 🇺🇦, БюджетOK 🇷🇺 | Автоматизація (Apple Pay, AI-скан) + синк, що «просто працює» без Dropbox/акаунтів |
| **Комбайни з банк-синком** | Saldo 🇺🇦, Fineye 🇺🇦, ZenMoney 🇷🇺, Wallet, Spendee, MoneyWiz | Надійність (без сервера = без 503) + приватність (без акаунта, без передачі банк-логінів) |
| **Нова AI-хвиля** | Finerd 🇺🇦, Qalta | «AI, що думає, а не лише слухає»: бюджети з AI + приватність iCloud (Finerd = все в OpenAI без бюджетів; Qalta = AI лише як швидкий ввід) |

**Наша унікальна позиція (перетин, який не займає ніхто):**
безкоштовний нативний iCloud-синк без акаунта (єдині з 15) + Apple Pay-автоматизація + AI (скан, категоризація, бюджет-розподіл) + справжні бюджети по категоріях + укр. розробник. Формула меседжу: **«Автоматичний облік без банку, без сервера і без реклами — твої дані лише у твоєму iCloud».** За позиціюванням (memory): вести з Apple Pay-автоматизації, приватності, AI-бюджетування; НЕ вести зі скану чеків.

## 3. Контент-план блогу

**Рішення «пост на кожну апку чи на декілька»: кластерне ядро СПОЧАТКУ, повне покриття ПОТІМ.**

## 6. Правила (чого НЕ робити)

1. **Не атакувати «своїх» брудно** (Saldo, Fineye, Sumlio, Finerd — українці): тільки чесні порівняння фактами. Меседж «підтримати своїх можна без компромісів».
2. **Не вести цінову війну** (Orange dog безкоштовний, Wallet lifetime $29.99, Saldo $2.49/міс — не переб'ємо).
3. **Не грати privacy-кутом проти Expenses і Qalta** — їхні лейбли чистіші за наш (Sentry). Проти них — глибина фіч.
4. **Не обіцяти банк-синк**, якого немає. Наша відповідь — Apple Pay-автоматизація + AI-імпорт виписок.
5. **Не порівнювати соц. докази** (у нас 28 оцінок проти тисяч) — не згадувати кількість оцінок у своїх матеріалах, поки не виростемо.
6. **Не вести маркетинг зі скану чеків** (позиціювання; світ безпаперовий).
7. У постах — жодної російської; цитати рос. відгуків — переказ українською.
